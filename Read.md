# Maven Market Sales Analysis

## Overview
This Power BI project analyzes Maven Market sales data to identify business trends, customer behavior, product performance, and regional sales performance.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV

## Dataset
The project uses Maven Market transactional data including:
- Customers
- Products
- Stores
- Regions
- Sales Transactions
- Returns

## Key Insights
- Sales Performance Analysis
- Customer Segmentation
- Product Performance
- Regional Analysis
- Return Rate Analysis
